Analog Discovery Tutorials

Voltage Instrument:

  • Project 1: Light an LED

Voltmeter Instrument:

  • Project 1: Measure voltages in Voltage Instrument tutorial

Waveform Generator:

  • Project 1: Waveform generator basics. Make an LED flicker.
  • Project 2: Sinusoidal sweeps. Listen to a frequency-varying sinusoid.
  • Project 3: Modulated sinusoids. Make some annoying siren sounds.
  • Project 4: Playing Audio (.wav) files. Some bird song .wav files which are appropriate for this project can be found here.
  • Project 5: Importing waveforms. Play "scales" using the Discovery.
  • Project 6: Creating signals from mathematical expressions and "beating".

Oscilloscope:

  • Project 1: Oscilloscope basics. Manually turn voltage on and off and measure response.
  • Project 2: Oscilloscope triggering. Use the AWG to make an LED flicker and use triggered acquisition to view the voltages.
  • Project 3: Oscilloscope measurements and cursors. Use a measurement window and cursors to display parameters for the circuit used in the second scope project.
  • Project 4: Math channels. Use a math channel to display the current in the circuit from the second scope project. (Continuation from the third scope project.)
  • Project 5: XY plots. Create a voltage-current curve for a diode. (Continuation from the fourth scope project.)
  • Project 6: Exporting data. Export the data from the fifth scope project to another application program. (Continuation from the fifth scope project.)
  • Project 7: Single sequence acquistion. Acquire vibration data from a piezoelectric sensor.
